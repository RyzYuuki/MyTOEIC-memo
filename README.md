# 📝 TOEIC 解答メモ＆タイマー

TOEICの学習を効率化するための解答メモアプリです。問題数を設定し、解答の選択肢を記録できます。さらに、タイマー機能も搭載しています。

## 🚀 主な機能

1.  **問題生成**: 任意の数の4択問題の入力欄を自動生成します。
2.  **解答記録**: A, B, C, D の選択肢をタップして解答を記録できます。
3.  **採点機能**: 各問題に「マル（OK）」または「バツ（NG）」を付けて採点し、結果を一覧で確認できます。
4.  **カスタマイズ可能なタイマー**:
    * 表示されている時間（例: `60:00`）を直接クリックして編集できます。
    * タイマーがゼロになると、アラートとともに音が鳴り始めます。
    * タイマーはスクロールしても画面上部に固定されます。
5.  **レスポンシブデザイン**: スマートフォンやタブレットの画面幅に応じて、選択肢や採点ボタンが自動で縮小し、画面からはみ出さないように最適化されています。

## 💻 使い方

1.  **問題数の設定**: アプリ上部の「問題数」フィールドに、生成したい4択問題の組数を入力します。
2.  **問題の生成**: 「生成」ボタンを押すと、入力した数だけ解答欄が表示されます。
3.  **解答**: 各問題のA/B/C/Dをタップして解答を記録します。
4.  **タイマー操作**:
    * 時間表示をクリックして、好きな時間に編集し、リセットボタンで確定します。
    * 「タイマー開始」ボタンでカウントダウンを開始/一時停止できます。
    * 「リセット」ボタンで最後に設定した時間に戻せます。
5.  **採点**: 問題の横にある「マル（OK）」または「バツ（NG）」をタップして採点します。
6.  **結果確認**: 画面下部の「解答一覧」エリアで、記録した解答と採点結果を確認できます。
